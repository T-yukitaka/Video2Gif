# Video2Gif

videoをgifに変換する．

## Prerequisites
- Linux or macOS
- python 3

## Getting Started
### Instalattion
- Clone this repo:
```
git clone https://github.com/T-yukitaka/Video2Gif.git
cd Video2Gif
```

- Conda Environment Setting

```
conda env create -n Video2Gif -f environment.yml
pip install -r requirements.txt
```

- Convert video to gif:
```
python Video2Gif.py
```
